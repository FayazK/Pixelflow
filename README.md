# pixelflow

An Electron application with React

## Packages
- [Better IPC](https://github.com/sindresorhus/electron-better-ipc)
- [Electron Store](https://github.com/sindresorhus/electron-store)

## Project Setup

### Install

```bash
$ yarn
```

### Development

```bash
$ yarn dev
```

### Build

```bash
# For windows
$ yarn build:win

# For macOS
$ yarn build:mac

# For Linux
$ yarn build:linux
```
